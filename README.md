## ğŸ¯ What Is This?

This project started as a creative sandbox with no specific end goal. It currently features:

- **Archery Mechanics**: Draw, aim, and fire arrows with physics-based trajectories
- **Multiplayer Support**: LAN and internet multiplayer with synced arrow visibility
- **Course Builder Tools**: Place objects and build environments
- **Open World Exploration**: Walk around freely in various test environments
